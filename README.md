# Marketing Dashboard Suite

## Overview

Developed multiple dashboards to support **sales performance tracking, campaign analysis, content performance monitoring, and order management**.  

Each dashboard was designed for a specific business function, enabling teams to make faster and more accurate decisions using **Salesforce data, GAM (Google Ad Manager), Google Analytics (GA), and Adbook**.

The dashboards focused on improving **revenue visibility, campaign performance tracking, product analysis, and content-level insights** while ensuring better coordination between sales, marketing, and operations teams.
